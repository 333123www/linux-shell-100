## 【题目要求】

写一个脚本实现如下功能：
输入一个数字，然后运行对应的一个命令。

显示命令如下：

*cmd meau**  1 - date 2 - ls 3 - who 4 - pwd
当输入1时，会运行date, 输入2时运行ls, 以此类推。

知识要点： case、read、if

习题解答：

```
#!/bin/bash 
function menu {
echo -e "***********MENU**********"
echo "1 - date"
echo "2 - who"
echo "3 - free"
echo "4 - break"
read -p "option:" -n 1 option
echo 
}
menu
while :
do
	case $option in 
	1)
        	echo 
        	date
        	echo;;
	2)
        	echo 
        	who
        	echo;;
	3)
        	echo 
        	free 
        	echo;;
	4)
        	echo 
        	break;;
	*)
        	echo 
        	echo "wrong!!!"
        	echo;;
	esac
	menu
done
echo "bye bye~~~"
```
