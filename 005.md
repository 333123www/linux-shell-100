## 【题目要求】

找到/123目录下所有后缀名为.txt的文件

  1. 批量修改.txt为.txt.bak
  2. 把所有.bak文件打包压缩为123.tar.gz
  3. 批量还原文件的名字，即把增加的.bak再删除

知识要点：tar、for、find

习题解答：

```
#!/bin/bash 

find 123/ -name *.txt >> txt.list 
for name in $(find 123/ -name *.txt)
do 
	mv $name $name.bak
done
tar -zcf 123.tar.gz 123/*.txt.bak
echo "123.tar.gz: "
tar -tf 123.tar.gz
for name1 in $(cat txt.list)
do
	mv $name1.bak $name1
done 
rm -rf txt.list
```
