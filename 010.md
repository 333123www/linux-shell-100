## 【题目要求】

用shell打印下面这句话中字母数小于6的单词。
Bash also interprets a number of multi-character options.

知识要点： for 、wc

习题解答：

```
#!/bin/bash 
str="Bash also interprets a number of multi-character options"
for num in $str
do
	a=`echo $num | wc -L`
	if [ $a -lt 6 ]
	then 
	    echo $num : $a
	fi
done

```
